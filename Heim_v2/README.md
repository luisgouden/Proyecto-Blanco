# Heim – Documentación de trabajo

Este repositorio recoge el worldbuilding y la estructura de la saga ambientada en Heim.

- Todo está pensado como documentación de autor, no como texto final para el lector.
- Los archivos están numerados para mantener orden lógico, no de publicación.

Carpetas principales:
- `10_heim_core/`: visión general y temas.
- `20_cosmologia/`: Aor, Thae-Ra, Demiurgo, Eones, Thymion…
- `30_magia/`: materia blanca, sellos, mundo blanco, portales.
- `40_culturas/`: Orka, Teeli, Akro/Lunari, Henso, Mieru, Velkorae.
- `50_eras/`: Atlantis y las distintas eras históricas.
- `60_personajes/`: fichas de personajes.
- `70_trama/`: estructura de libros, arcos y capítulos.
- `80_sistema/`: reglas operativas (identidad en mundo blanco, economía, etc.).
